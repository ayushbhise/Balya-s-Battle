🚀 About the Game

Balya’s Battle is a fast-paced browser-based action game where players control Balya and fight through challenges using quick reflexes and timing.

Built as a lightweight web game, it runs directly in your browser with no downloads required.

🕹️ How to Play
Use on-screen controls (or keyboard if supported)
Move your character to avoid enemies or obstacles
Progress by reacting quickly and staying alive
Timing matters — play smart, not just fast
⚡ Features
Instant play (no installation)
Lightweight and fast loading
Simple controls, hard gameplay
Designed for quick fun sessions
